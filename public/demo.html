Raw API

Base URL: /api

  • POST /api/chat/init { ip?, userId? } → session
  • POST /api/chat/end { sessionId }
  • POST /api/chat/message { sessionId, message } → { reply }
  • GET /api/chat/history/:sessionId → { messages }
  • GET /api/chat/suggest?limit=5 → { suggestions }
  • GET /api/faq → [ { id, question, answer }, ... ]
  • POST /api/faq { question, answer }
  • PUT /api/faq/:id { question?, answer? }
  • DELETE /api/faq/:id
